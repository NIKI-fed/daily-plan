# React + Vite + JS

## Installation

1. Fork on Github
2. git clone git@github.com:YOUR-PROFILE/TTN on terminal
3. npm i - установить зависимости
4. npm run dev - запустить сервер разработки
5. npm run build - для сборки в папку dist
6. npm run preview - проверка сбилженного проекта из папки dist
